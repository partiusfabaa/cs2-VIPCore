# Adds bunnyhop for VIP players

# Config

### in vip.json
```json
"Bhop": {
  "Timer": 5.0, // where 5.0 is how long after the start of the round the bhop will be available. You can write as much as you want.
  "MaxSpeed": 350 // maximum jump speed
}
```
